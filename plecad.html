Plataforma de Evaluación de Calefacción Distrital

Descripción

La Plataforma de Evaluación de Calefacción Distrital (PLECAD) es un proyecto Fondo de Innovación para la Competitividad FIC-R financiado por el Gobierno Regional del Biobío, y consiste en una plataforma web que permitirá a los usuarios evaluar la posible implementación de un sistema de calefacción distrital según las características de su localidad. Su director (A) es el Dr. Eduardo Espinosa Neira.

Desarrollo

La plataforma web se encuentra en desarrollo y se espera que esté disponible para su uso en el segundo semestre de 2024. Las funciones backend realizadas en este proyecto son:

Hitos:

  • Conexión de los dispositivos sensores IQAir indoor y outdoor.
  • Configuración de la API de cada dispositivo sensor.
  • Desarrollo de la base de datos en BigQuery (GCP) para almacenar los datos de los sensores.
  • Extracción de los datos de los sensores, para su posterior almacenamiento en BigQuery a través de Python.
  • Desarrollo de gráficos en Python a partir de los datos obtenidos por los sensores.
  • Desarrollo de función para determinar el Índice de Calidad del Aire (AQI US) en Python.
  • Desarrollo de iframe en Python con gráfico a insertar en la plataforma.

  • El desarrollo de la plataforma web es independiente a las funciones realizadas por mí. La empresa Puchaki está a cargo del desarrollo de ésta.

Tecnologías utilizadas:
  • API IQAir
  • Python (Pandas, Numpy, Matplotlib, Plotly, entre otras librerías)
  • Google Cloud Platform (GCP)
  • BigQuery
  • Cloud Functions (Crontab)
  • WordPress
  • Debian
